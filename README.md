# Project One Roofing Proposal

Premium interactive proposal website for Ghost AI Solutions to position the company as Project One Roofing's creative capture and marketing amplification partner.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP
- Lucide Icons

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production Check

```bash
npm run lint
npm run build
```

## Note

The Ghost AI Solutions logo is included at `public/ghost-ai-logo.png`.
